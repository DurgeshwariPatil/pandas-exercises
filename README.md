# Kaggle Pandas Course - Exercise Notebooks

This repository contains my solutions for the Kaggle Pandas course exercises. 
It includes all 6 Jupyter notebooks covering different topics in data analysis using Pandas.

## Contents
The notebooks include practice on:
- Creating, reading and writing
- Indexing, selecting and assigning
- Summary functions and maps
- Grouping and sorting
- Data types and handling missing values
- Renaming and combining data

## Purpose
These notebooks helped me understand:
- Data manipulation
- Data cleaning
- Handling missing values
- Filtering and grouping data
- Real-world data analysis workflows

## Tools Used
- Python
- Pandas
- Jupyter Notebook / Kaggle Notebooks

## About the Course
The Kaggle Pandas course provides hands-on exercises designed to improve practical data analysis skills and build a strong foundation for advanced analytics and machine learning.
